Design Concept

Bin.Go! is a desktop user interface that helps control and run the system of autonomous garbage robots that go from house-to-house and collects disposable waste. Through its autonomous functions, this robot will be able to solve all the problems that current residents have in regards to garbage collection. From user research, these problems included inconsistent garbage collections times, traffic issues, not being eco-friendly, is inconvenient, and produces noise pollution. The user of this desktop interface are council workers related to managing garbage collection and its system.

Throughout the process, we made various important design decisions that helped define the design of the current user interface. These include:
Static Navigation Bar - We made the navigation bar on the left to be static because it is able to assure the user about the different tabs that the webpage has.
Colour Theme - The colour palette of the website was to be minimalistic and modern, this was done through the contrast of different shades of grey (light and dark). The setting of the dark mode is able to change that though, which allows for user freedom.
Navigational Map - This was used so that it would allow for more interaction between screen and user. It also allows for a more fluid tracking of the autonomous robots that are out.
Icons - Icons are used so that users are able to easily and quickly able to identify content without having to focus their mind only on reading.

Reflection

Throughout the whole process of creating Bin.Go!, I have definitely grown a lot as a designer. I have learnt new skills in this journey. These include the ability to use different softwares in creating an interactive application (e.g Framer, InVision, Figma), and to be able to identify the correct context in using each different method of user research.
In addition to learning new skills, I was able to refine existing designer skills. These include the confidence to work in a collaborative environment, the capability to identify and design a product according to user needs & wants, and to be able to think more critically and extensive.
Overall, I have developed myself greatly as a designer through the creation of Bin.Go! and my fire of passion for designing products is burning ever so brightly!
